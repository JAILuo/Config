# theme

ubuntu22.04 + 默认 gnome terminal...lazy :)

使用 Dracula 的颜色主题，不过多解释，官网有步骤，跟着一步步做即可：

[Gnome Terminal • Dracula Theme](https://draculatheme.com/gnome-terminal)

唯一需要注意的是，对于终端的Preferences，我另外开了一个 profile 来保存配置（我自己的叫 `dracula`）

这也是安装的主题时使用的默认的配置，其中相关命令：

```shell
sh install.sh -s Dracula -p dracula --skip-dircolors
```
